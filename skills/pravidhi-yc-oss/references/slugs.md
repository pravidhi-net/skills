# YC OSS API Taxonomy

Use these specific slugs for the `--target` argument in `yc_client.py`. 
**Do not guess.** If a term is not listed here, use `--mode search --keyword "Term"` instead.

## Batches (Partial List)
Format: `[season][year]` (lowercase)
- `w24`, `s23`, `w23`, `s22`, `w22`
- `s21`, `w21`, `s20`, `w20`
- ... back to `s05`
